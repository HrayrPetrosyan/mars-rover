# Mars Rover

## Tech stack

| Field | Technology |
| ------ | ------ |
| Language | TypeScript |
| Runtime env | Node JS |
| Unit tests | Jest |
| Version control | Git |

## Installation

Mars Rover requires [Node.js](https://nodejs.org/) v10+ and [Yarn](https://yarnpkg.com/) to run.

Install the dependencies and devDependencies and start the server.

```sh
yarn
yarn start
```
